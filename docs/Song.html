toString( [format])

Produces a string based on given formatting. The default formatting is:

%artist - %title (%remix)

The information is extracted from the tags stored in the database. If a tag is empty an empty string is returned.

Note that the keywords are case-sensitive.

Keywords that can be used:

%artist
%title
%remix
%year
%album
%label
%genre
%trackNumber
%composer
%bpm
%key
%grouping
%stars
%user1
%user2
